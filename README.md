# Rusty Hearts UI Editor

[English version](README_EN.md)

Визуальный редактор UI для игры Rusty Hearts. Позволяет просматривать и редактировать интерфейсы игры с поддержкой переводов.

![Screenshot](screenshot1.png)

## Возможности

- **Просмотр UI окон** — загрузка и отображение XML файлов интерфейса
- **Рендеринг спрайтов** — поддержка всех типов спрайтов (static, button, 9-slice, gauge, scroll и др.)
- **Работа с переводами** — загрузка переводов из XLSX файлов, редактирование в реальном времени
- **Поддержка тегов** — корректное отображение `<BR>`, `<color:RRGGBB>`, `</color>` и других игровых тегов
- **Редактирование** — изменение позиции, размера, шрифта элементов
- **Определение переполнения** — визуальная индикация текста, выходящего за границы элемента
- **Мультиязычность** — интерфейс на русском и английском языках

## Установка

1. Установите [Node.js](https://nodejs.org/) (версия 16+)

2. Клонируйте репозиторий:
```bash
git clone https://github.com/your-repo/rusty-hearts-ui-editor.git
cd rusty-hearts-ui-editor
```

3. Установите зависимости:
```bash
npm install
```

4. Настройте пути в `server.js`:
```javascript
const CONFIG = {
    UI_PATH: 'путь/к/ui/файлам',
    TRANSLATIONS_PATH: 'путь/к/переводам',
    RESOURCES_PATH: 'путь/к/ресурсам'
};
```

5. Запустите сервер:
```bash
npm start
```

6. Откройте в браузере: http://localhost:3847

## Использование

1. Выберите окно из выпадающего списка
2. Кликните на элемент для выбора
3. Редактируйте свойства в правой панели
4. Для редактирования перевода перейдите на вкладку "Перевод"
5. Изменения текста отображаются в реальном времени
6. Нажмите "Сохранить" для сохранения изменений

### Горячие клавиши

- `Ctrl+S` — сохранить изменения
- `Escape` — снять выделение
- `Стрелки` — перемещение выбранного элемента
- `Shift+Стрелки` — перемещение с шагом 10px
- `Ctrl+Колесо мыши` — масштабирование

## Поддерживаемые типы спрайтов

| Тип | Описание |
|-----|----------|
| static | Статичное изображение |
| button | Кнопка (3-4 состояния) |
| pushbutton | Нажимаемая кнопка |
| checkbutton | Чекбокс |
| gauge | Прогресс-бар |
| v_scroll | Вертикальный скроллбар |
| combo | Выпадающий список |
| edit | Текстовое поле |
| 9-slice | Растягиваемый фрейм |
